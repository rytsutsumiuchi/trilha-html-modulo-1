Como é uma estrutura básica de um HTML?

(Voltar)

A estrutura básica de um HTML é formada pelas tags <html>, <head> e <body> no seguinte formato:

      <html>   
            [Essa tag é o container para todos os outros elementos HTML]
      <head>   
            [Essa tag contém os dados sobre o arquivo HTML como o título da página, estilos usados, scripts entro outros]
      </head>  
      <body>   
            [Essa tag contém o corpo do documento. Tudo o que é exibido para o usuário fica aqui]
      </body>
      </html>

Entendendo a diferença entre listas ordenadas e não ordenadas

(Voltar)

Existem dois tipos de listas, as ordenadas (ol - ordered list) e as não ordenadas (ul - unordered list).

A primeira é usada quando se quer apresentar uma ordem na lista, que pode ser por número ou letra.

  1. Item 1
  2. Item 2
  3. Item 3
  1. Item 1
  2. Item 2
  3. Item 3

A segunda não indica uma ordem, e cada item só é precedido por um ponto.

  • Item
  • Item
  • Item

A estrutura de listas é a seguinte:

      <ol>              [<ul> no caso de listas não ordenadas]
         <li>           [São os list items ou itens da lista]
            Item 1      [Cada item deve ser inserido dentro de uma tag <li></li>]
         </li>
         <li>
            Item 2         
         </li>
         <li>
            Item 3         
         </li>
      </ol>           [</ul> no caso de listas não ordenadas]
